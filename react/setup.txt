1. npx install-peerdeps --dev eslint-config-airbnb

//order is important here, plugins like being installed at the end:
2. npm i -D eslint-config-prettier eslint-plugin-styled-components-a11y eslint-plugin-jsdoc eslint-plugin-html

3. npm update (npm outdated will show out of date packages)

5. restart VSCode

**Eslint version may not be latest and will throw an error with react-scripts. npm i -D eslint@latest
**If eslint errors from throwing in the browser, create a .env file in the root folder with the flag ESLINT_NO_DEV_ERRORS=true. add .env to .gitignore
